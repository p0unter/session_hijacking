# Cookie Paths

### Firefox/Firefox-Based

```bash

    Windows: C:\Users\<your_username>\AppData\Roaming\Mozilla\Firefox\Profiles\<random list of characters>.default\cookies.sqlite // Firefox

    Windows: C:\Users\<your_username>\AppData\Roaming\<name of your browser>\Profiles\<random list of characters>.default\cookies.sqlite // Firefox-Based (Floorp,LibreWolf,Zen etc.) 

    (In some browsers the Profile name is like this: <random list of characters>.default-release)

    Linux: UNKNOWN

    macOS: UNKNOWN   

```

### Chrome/Chromium

```bash

    Windows: C:\Users\<your_username>\AppData\Local\Google\Chrome\User Data\Default\Network
    Linux: ~/.config/chromium/Default/Cookies
    macOS: ~/Library/Application Support/Google/Chrome/Default/Cookies

```

### Brave

```bash

    Windows: C:\Users\<your_username>\AppData\Local OR AppData\Roaming
    Linux: /.config/BraveSoftware/
    macOS: /Users/<your_username>/Library/Application Support/BraveSoftware/

```

### Vivaldi

```bash

    Windows: C:\Users\<your_username>\AppData\Local\Vivaldi\User Data\Default\Network
    Linux: UNKNOWN
    macOS: UNKNOWN

```
